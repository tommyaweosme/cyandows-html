About Cyandows HTML

Cyandows HTML is an offline OS belonging to Cyansoft (link leads outside of OS). The OS works entirely offline. It is reccomended to fullscreen the tab for the best experience.

How Chrome works offline?

It uses procedural generation to simulate over 1 billion unique websites you can visit.

Best practices on Notepad

Since there is no way to view all your files, keep a directory of all your files on the blank file.

Best practices on Minesweeper

Because of the window size, 12x21 is the maximum size Minesweeper grids can be before they get unplayable.

History of Cyandows HTML

I didn't like Chrome OS on my computer so I made a simple HTML OS and just kept adding to it. The main issue was that there were no icons on the wallpaper, like that is the defining feature of an OS how can you NOT have it!!!
