# cyandows-html
An operating system by my fictional company, Cyansoft, that runs in-browser.
Load up "index.html" for the full experience or "os.html" to boot directly into the OS. Fullscreen is heavily reccomended. Designed for Chrome on Chrome OS.
## Apps
Notepad
Minesweeper
OS Info
Calculator
Cyancraft
Chrome
Paint
Cyandows VM
Music Player
