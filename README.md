# cyandows-html
An operating system by my fictional company, Cyansoft, that runs in-browser.
Load up "index.html" for the full experience or "os.html" to boot directly into the OS. Fullscreen is heavily reccomended. Designed for Chrome on Chrome OS. Please report any and all bugs to me. The directory here may not be fully up-to-date with the directory on my computer.
